## 9️⃣ Анимации

### fadeIn (появление элементов):

```
0%   → opacity: 0, translateY(10px)
100% → opacity: 1, translateY(0)
```

### shake (ошибка):

```
0%   → translateX(0)
25%  → translateX(-5px)
75%  → translateX(5px)
100% → translateX(0)
```

### aurora (фоновый эффект):

```
0%   → translate(0, 0) scale(1)
33%  → translate(30px, -30px) scale(1.1)
66%  → translate(-20px, 20px) scale(0.9)
100% → translate(0, 0) scale(1)
```

## 🎯 Ключевые улучшения UX

1. **Понятный flow** - пользователь сразу видит кнопки "Войти" и "Регистрация"
2. **Информативность** - подсказки объясняют, зачем нужна регистрация
3. **Валидация** - ошибки показываются в реальном времени
4. **Обратная связь** - кнопки меняют текст при загрузке
5. **Гостевой доступ** - друзья могут присоединяться без регистрации
6. **Безопасность** - JWT токены, bcrypt пароли
7. **Современный дизайн** - glass morphism, градиенты, анимации
